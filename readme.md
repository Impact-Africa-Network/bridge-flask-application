# how to run the project
# create a virtual env and activate it
# install requirements

```shell
pip install -r requirements.txt
```

# create the db

```shell
python db.py
```

# run the project in debug mode

```shell
flask  --debug run
```



install jwt

```shell
pip install jwt
```

install sqlalchemy

```shell
pip install sqlalchemy
```

```shell
pip install bcrypt

```
